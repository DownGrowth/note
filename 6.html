JS基本语法

表达式

  • 1+2是一个表达式，值是3
  • add(1+2)表达式的值为函数的返回值
  • console.log表达式的值是函数本身
  • console.log(3)的值是undefined

语句

  • var a=1是一个语句
  • 大部分空格都没有意义，回车也是，但在var和a中间空格不能省略，return后面也不能加回车

标识符

  • var后面可以跟Unicode字母或$或_或中文，但第一个字符不能是数字

if else语句

  • 用法：if(表达式){语句1}else{语句2}
  • 语句1和语句2都可以嵌套if else
  • 逗号表示一句话没有结束，分号表示结束
  • 最推荐的写法if(表达式){

  • 如果if else里只有一个语句，就用?:表达式。用法是：表达式1? 表达2: 表达式3”
  • switch语句 switch(fruit){

    case "banana":

    // ...

    break;

    case "banana":

    // ...

    break;

    default:

    // ...

    }
  • &&短路逻辑，A&&B&&C&&D，如果A为真则输出D，A为假则输出A
  • ||短路逻辑，A||B||C||D,A为假则输出D，A为真则输出A，与上面的相反。

while、for循环

  • var a=1 (初始化) whlie(表达式，就是判断条件){语句,就是循环体}
  • 判断表达式的真假
  • 当判断式为真，执行语句，执行完再看表达式的真假
  • 当判断式为假，执行后面的语句
  • 浮点数会不精确
  • for是while的方便写法
  • for(语句1；表达式2;语句3){循环体}
  • 先执行语句1
  • 然后判断表达式2
  • 如果为真，执行循环体，然后执行语句3
  • 如果为假，则跳出循环体，执行后面的语句
  • 上述for语句中的四个部分都可以省略，但可能会缺条件而一直循环
  • setTimeout是过一会再执行

break和continue

  • break是退出当前的所有循环，continue是退出这次的循环，去下一次的
