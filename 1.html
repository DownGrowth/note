《HTML入门笔记1》

  1. html是李爵士发明的
  2. HTML 起手应该写 " !DOCTYPE html html lang="zh-CN" head meta charset="UTF-8" / meta name="viewport" content="width=device-width,initial-scale=1.0" / meta http-equiv="X-UA-Compatible" content="ie=edge" / title我的网页/title /head body/body /html"
  3. 常用的表章节的标签有： 
              h1~h6(1-6级标题）;
          section（章节）;
          article（文章）;
          p（段落）;
          main（主要内容）;
          header（头部）;
          footer（脚部）;
          aside（旁支内容）;
          div（划分）;
  4. 全局属性有: 
                class;
            contenteditable;
            hidden;
            id;
            style;
            title;
            tabindex;
  5. 常用的内容标签有哪些: 
                ol+lo （按顺序列出）;
            ul+li （无序列出）;
            dl+dt+dd （描述内容）;
            pre （加空格）;
            hr （水平分割线）;
            br （换行）;
            a （超链接）;
            em （语气上强调）;
            strong （内容上强调）;
            code （代码）;
            quote （引用）;
            blockquote （块的引用）;
